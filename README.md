# CmdEnter

Muscle memory for the tools you live in. A keyboard-fluency trainer for 24 apps —
Cursor, Linear, Excel, Gmail, AutoCAD, Rhino, Revit, Grasshopper, and more.

Live: https://cmdenter.app

Single self-contained static site (`index.html`) — vanilla JS, no build step,
no backend. License validation via Lemon Squeezy's public API; email capture via
Formspree. Deployed on Vercel from this repo.

The core decks are free. See the app for pricing.
